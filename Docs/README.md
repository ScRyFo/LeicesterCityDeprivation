# Leicester City Deprivation Analysis

This project analyses IMD 2025 data for Leicester using a reproducible R
pipeline.

## Key Features

-   Ward-level aggregation of LSOA data
-   Z-score standardisation for comparability
-   Multi-domain analysis across IMD domains
-   Fully reproducible workflow

## How to run

1.  Place IMD dataset in `/data`
2.  Open `scripts/imd_pipeline.R`
3.  Modify parameters (year, domain, wards)
4.  Run script

## Outputs

-   Comparative ward charts
-   Domain-level deprivation insights

## Data Source

Leicester City Council Open Data Portal
